# Imágenes del Portafolio

Coloca aquí las imágenes de tu portafolio. A continuación las instrucciones:

## og-image.png (REQUERIDO)
- **Ubicación**: `/public/og-image.png`
- **Dimensiones**: 1200 x 630 px
- **Formato**: PNG o JPG
- **Propósito**: Imagen de previsualización cuando compartan el link del portafolio en redes sociales (Open Graph)
- **Recomendación**: Incluye tu nombre, título profesional y un diseño limpio con alto contraste

## Imágenes adicionales (OPCIONAL)
Si deseas agregar imágenes de proyectos, logos de clientes, o foto personal:

| Archivo | Dimensiones Recomendadas | Formato | Uso |
|---------|-------------------------|---------|-----|
| profile.jpg | 400 x 400 px | JPG/WebP | Foto personal (si agregas sección about) |
| project-*.jpg | 800 x 600 px | JPG/WebP | Screenshots de proyectos |
| logo.svg | 40 x 40 px | SVG | Logo personal para navbar |

### Tips de optimización:
- Usa formato WebP cuando sea posible (mejor compresión)
- Comprime las imágenes antes de subir (tinypng.com)
- Las imágenes grandes impactan el tiempo de carga — mantén cada archivo < 200KB
